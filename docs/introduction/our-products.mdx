The Swarm Corporation provides three core products: 

> **Swarms API** for building multi-agent systems

> **ATP Protocol** for enabling economic transactions between agents

> **Swarms Marketplace** for discovering, creating, and monetizing agents and prompts.

Payment settlement system for AI agent services on Solana blockchain. Enables automatic, usage-based payments with sub-second transaction times.

<Card title="Usage-Based Payments" icon="calculator" href="/docs/atp/overview">
